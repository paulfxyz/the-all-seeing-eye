👁 How It Works

The All Seeing Eye is a self-hosted, open-source uptime, DNS, SSL and latency monitor. Everything runs in the browser — no server-side code needed for basic use.

📄 domains.list
One domain per line. Lines starting with # are comments. Loaded at startup — if missing, the built-in top-30 list is used. Edit this file to change what gets monitored.
📊 domains.stats
CSV snapshot written after every refresh cycle. Requires a server with HTTP PUT write access. On static hosts, the export CSV button gives the same data as a manual download.
🔗 webhook.do
Headless endpoint for cron services. When accessed, it loads the dashboard in an invisible iframe, runs all DNS checks, and writes domains.stats — no browser or PIN required.
🌐 DNS Checks
Each domain is queried via Cloudflare DoH (HTTPS, no CORS issues). An A-record response = online. Latency = full round-trip time. Auto-refreshes every 3 minutes.
🔐 PIN Security
PIN is stored as a SHA-256 hash only — never plaintext. The hash is checked client-side. No data is sent anywhere. Change the PIN by recomputing sha256(yourPin) and updating PIN_HASH in the script.
