What ASE does

Every signal,
all at once.

From A records to DMARC policies, SSL expiry to uptime streaks — ASE sees everything your domains are whispering, and tells you before they shout.

Live uptime monitoring
A-record checks via Cloudflare DoH. 5 parallel queries per domain. Batch-of-5 scanning to avoid DoH flood detection. Latency to the millisecond. Uptime history persisted server-side.
SSL certificate tracking
Real TLS handshakes via server-side PHP. Batch mode covers all domains in one request. Expiry warnings at 30d and 7d. Issuer detection — automatic Let's Encrypt badge. Zero false positives.
DMARC · SPF · MX health
Full email security posture per domain. Detects missing DMARC, p=none policies, missing SPF. NS and MX provider auto-identified and labelled. Email deliverability at a glance.
Smart email alerts
Digest emails via Resend API with full domain health snapshot. Dual cooldown: 5-minute on manual refresh, 24-hour on auto. AES-256-GCM encrypted API keys. One email per event — never a flood.
Beautiful dashboard
Progressively-rendered table with real-time stats. Dark/light mode, CSV export, per-row refresh. Mobile-first with native numeric PIN keyboard. No framework, no npm, no build step.
Open source · self-hosted
MIT licensed. Your data stays on your server. No accounts, no cloud dependencies, no tracking. Three files to deploy. Works on SiteGround, cPanel, Cloudflare Pages — everywhere PHP runs.

How it works

Up in minutes.
On guard forever.

1
Upload three files
Drop index.html, app.css, and app.js onto any web host. No npm, no build step, no Composer. Works on SiteGround, cPanel, or static hosting in 60 seconds.
2
Add your domains
Edit domains.list — one domain per line. Or add live via the dashboard. Checks fire immediately: A records, NS, MX, TXT, DMARC, SPF, SSL.

Auto-refresh every 3 minutes. Cron optional (10-min server-side PHP). Email alerts via Resend on downtime, SSL expiry, or missing DMARC — with smart cooldowns to prevent inbox spam.

ssl-check.php — TLS Handshake
// Real TLS handshake — no curl shortcuts function checkSSL($domain) { $ctx = stream_context_create([ 'ssl' => [ 'capture_peer_cert' => true, 'verify_peer' => true, 'verify_peer_name' => true, ] ]); $socket = stream_socket_client( "ssl://{$domain}:443", $errno, $errstr, 10, STREAM_CLIENT_CONNECT, $ctx ); $cert = stream_context_get_params($socket); $info = openssl_x509_parse( $cert['options']['ssl']['peer_certificate'] ); return [ 'expiry' => $info['validTo_time_t'], 'issuer' => $info['issuer']['O'], 'days' => daysUntil($info['validTo_time_t']) ]; }
encryption.php — AES-256-GCM Key Storage
// Encrypt API keys at rest function encrypt($plaintext, $pin) { $key = hash('sha256', $pin, true); $iv = random_bytes(12); $tag = ''; $data = openssl_encrypt( $plaintext, 'aes-256-gcm', $key, 0, $iv, $tag ); return base64_encode( $iv . $tag . $data ); }

Why AES-256-GCM? Your Resend API key and email address are encrypted at rest using your PIN as the passphrase. No plaintext secrets ever touch the filesystem.

Key derivation: SHA-256 hash of your numeric PIN produces the 256-bit encryption key. The 12-byte IV is random per encryption. GCM provides authenticated encryption — any tampering is detected.

Result: Even if someone accesses your server files, they can't read your API keys without the PIN. The PIN never leaves the browser.

⚡ Always-on monitoring

Persistent uptime records,
zero maintenance.

ASE writes every check to uptime.json on your server — building a cumulative history that survives browser closes, device switches, and midnight cron runs. Point a free webhook scheduler at webhook.do and your dashboard becomes a fully automated, always-recording uptime platform.

Schedule a free webhook
Use cron-job.org, UptimeRobot, GitHub Actions, or any HTTP scheduler. Hit yourdomain.com/webhook.do every hour — free, 30 seconds to set up, no server access needed.
↓
ASE checks everything
Each webhook call triggers a full DNS + uptime scan across all your domains — A records, NS, MX, DMARC, SPF, SSL expiry — exactly as if you had just hit Refresh. No browser tab needs to be open.
↓
Results saved to uptime.json
Every check — from any browser, device, or scheduled call — is merged into a single server-side file. Uptime percentage, first-seen timestamp, and last-down time accumulate indefinitely. Your 30-day SLA data is just a file on your server.
↓

Downtime, SSL expiry ≤30 days, missing DMARC, missing SPF — ASE fires a digest email via Resend the moment an issue is detected, with smart cooldowns to prevent flooding. One email. All issues. Every domain.
